# Inforsion Frontend Project

Inforsion 프론트엔드 프로젝트 리포지토리입니다.


## 프로젝트 설치하기

프로젝트를 설치하기 위한 방법입니다.
1. 터미널에 `git clone https://github.com/Inforsion/inforsion-fe.git` 입력해서 프로젝트 클론하기
2. `npm install`로 의존성 패키지 설치하기
3. `npm run dev`로 서버 작동하기
